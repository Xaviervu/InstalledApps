A test app for retreiving a list of apps with folowing details:
1. App Name
2. App Version
3. App package name
4. App SHA-1 checksum
5. App icon

A MVVM pattern was used for simplicity and reliability
Stack:
Coroutines, SharedFlow, StateFlow, Compose, ViewModel
